# wire-briefing

신선기(Wire Drawing Machine) · 열처리 설비 담당자를 위한 일일 브리핑 자동 생성 프로젝트.

## 개요

매일 오전 8시, 전세계(한국·일본·독일·영어권)에서 신선기·열처리 설비 관련 최신 정보를 수집하여 HTML 브리핑 파일을 생성한다.

## 수집 카테고리

| # | 카테고리 | 설명 |
|---|---|---|
| 1 | 설비 기술 동향 | 신선기·열처리로 최신 기술 (메인) |
| 2 | 자동화·무인화 | 로봇·AGV·자동 접속 등 |
| 3 | 품질·인라인 검사 | 측경·결함 검출 등 |
| 4 | 에너지 절감·예지보전 | 인버터·센서·진동 분석 등 |
| 5 | 업계·규격 동향 | KS·ISO 개정, 설비업체 뉴스 등 |

## 파일 구조

```
wire-briefing/
├── reports/          # 생성된 일일 브리핑 HTML (YYYY-MM-DD_wire_report.html)
├── templates/        # HTML 템플릿
└── CLAUDE.md         # Claude 동작 규칙
```

## 실행 방법

Claude Code 스케줄 태스크 **wire-briefing** 이 매일 오전 8시 자동 실행된다.  
수동 실행은 Claude Code에서 스케줄 태스크를 직접 트리거한다.

## 브리핑 예시

생성된 HTML 파일을 브라우저로 열면 카테고리별 카드 형태로 확인할 수 있다.
